# 🧠 Настройка моделей эмбеддингов

## Обзор

Система использует **локальные модели эмбеддингов** из библиотеки Sentence Transformers. Модели скачиваются один раз и затем работают полностью офлайн.

## Модели по умолчанию

- **Основная модель**: `paraphrase-multilingual-MiniLM-L12-v2`
  - Размерность: 384
  - Поддержка: Русский, Казахский, Английский и другие языки
  - Размер: ~420 MB
  
- **Запасная модель**: `all-MiniLM-L6-v2`
  - Размерность: 384
  - Быстрее, но менее точная для русского языка
  - Размер: ~90 MB

## Предзагрузка моделей

Для предзагрузки моделей перед первым запуском:

```bash
python3 preload_embeddings.py
```

Или запустите `start_system.sh` - он автоматически проверит и загрузит модели при необходимости.

## Использование GPU

Система автоматически определяет наличие GPU и использует его для ускорения вычислений эмбеддингов.

### Настройка через переменные окружения

В файле `.env` можно настроить:

```bash
# Использование GPU для эмбеддингов
# 'auto' - автоматически определять (по умолчанию)
# 'true' - принудительно использовать GPU
# 'false' - использовать только CPU
USE_GPU_FOR_EMBEDDINGS=auto

# Выбор модели эмбеддингов
EMBEDDING_MODEL=sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2
EMBEDDING_MODEL_OFFLINE=all-MiniLM-L6-v2
```

## Где хранятся модели

После загрузки модели сохраняются локально в:
- `~/.cache/huggingface/` или
- `~/.cache/torch/sentence_transformers/`

Модели не нужно скачивать повторно - они используются из кеша.

## Альтернативные модели для русского языка

Если нужна лучшая поддержка русского языка, можно использовать:

```bash
# В .env файле:
EMBEDDING_MODEL=cointegrated/rubert-tiny2
```

Или более точную (но тяжелее):

```bash
EMBEDDING_MODEL=DeepPavlov/rubert-base-cased-sentence
```

## Проверка работы

После загрузки модели вы увидите сообщения:
- `✅ Модель эмбеддингов загружена`
- `📱 Устройство: cuda` (если используется GPU) или `cpu`

## Устранение проблем

### Модель не загружается

1. Проверьте подключение к интернету (только для первой загрузки)
2. Проверьте свободное место на диске (нужно ~500 MB)
3. Попробуйте загрузить модель вручную:
   ```bash
   python3 -c "from sentence_transformers import SentenceTransformer; SentenceTransformer('sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2')"
   ```

### GPU не используется

1. Убедитесь, что установлен PyTorch с поддержкой CUDA:
   ```bash
   python3 -c "import torch; print(torch.cuda.is_available())"
   ```
2. Если False, установите PyTorch с CUDA:
   ```bash
   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
   ```

### Медленная работа

- Используйте GPU если доступно
- Используйте более легкую модель (`all-MiniLM-L6-v2`)
- Уменьшите размер батча в коде (если обрабатываете много документов)

## Производительность

Примерная скорость создания эмбеддингов:

| Устройство | Скорость (текстов/сек) |
|-----------|------------------------|
| CPU (Intel i7) | ~50-100 |
| GPU (NVIDIA GTX 1060) | ~500-1000 |
| GPU (NVIDIA RTX 3090) | ~2000-5000 |

*Зависит от длины текста и модели*
