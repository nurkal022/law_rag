# Быстрая инструкция по добавлению документов в git

## Текущая ситуация

✅ В директории `current/examples/` уже есть 10 примеров документов (240KB)
✅ `.gitignore` настроен для исключения основной директории `current/`, но разрешает `current/examples/`

## Добавление в git

```bash
# Добавить примеры документов
git add current/examples/

# Проверить что добавлено
git status

# Закоммитить
git commit -m "Add example documents for deployment"
```

## Если файлы не добавляются

Если git все еще игнорирует файлы, используйте принудительное добавление:

```bash
git add -f current/examples/*.txt current/examples/README.md
```

## Проверка размера

```bash
# Размер директории с примерами
du -sh current/examples/

# Количество файлов
find current/examples -name "*.txt" | wc -l
```

## Обновление примеров

Для обновления набора примеров используйте скрипт:

```bash
# Python версия (рекомендуется)
python3 scripts/prepare_examples.py

# Или bash версия
bash scripts/prepare_examples.sh
```
