# 🚀 Быстрый запуск системы

## Один командой

```bash
cd /home/kaznu2025/law_rag
./start_system.sh
```

## Что делает скрипт

1. ✅ Проверяет виртуальное окружение
2. ✅ Устанавливает зависимости (если нужно)
3. ✅ Запускает Fine-tuned API сервер (если доступен)
4. ✅ Запускает Flask приложение

## Доступ к системе

После запуска откройте в браузере:

- **Главная**: http://localhost:5003
- **RAG чат**: http://localhost:5003/chat
- **Простой чат**: http://localhost:5003/chat-simple
- **Админ**: http://localhost:5003/admin

## Остановка

```bash
./stop_system.sh
```

Или нажмите `Ctrl+C` в терминале где запущено приложение.
