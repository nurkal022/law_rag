# Примеры документов для развертывания

Эта директория содержит небольшой набор примеров документов для демонстрации работы системы и развертывания на сервере.

## Содержимое

В директории находятся примеры различных типов юридических документов Республики Казахстан:

- **Бюджетные документы** - решения о бюджетах различных уровней
- **Постановления Правительства** - нормативные акты правительства РК
- **Указы Президента** - указы главы государства
- **Решения маслихатов** - решения местных представительных органов
- **Комментарии** - комментарии к нормативным актам

## Использование

Эти документы используются для:
1. **Демонстрации** работы системы RAG
2. **Тестирования** функционала поиска и генерации ответов
3. **Развертывания** на сервере без необходимости загружать полный корпус документов

## Добавление документов

Для добавления большего количества документов:

1. Скопируйте файлы в эту директорию:
```bash
cp path/to/document.txt current/examples/
```

2. Или используйте скрипт подготовки:
```bash
bash scripts/prepare_examples.sh
```

## Важно

- Основная директория `current/` исключена из git через `.gitignore`
- В git попадают только файлы из `current/examples/`
- Для продакшена рекомендуется загрузить полный корпус документов через админ-панель

## Обработка документов

После развертывания на сервере:

1. Загрузите документы через админ-панель (`/admin`)
2. Или используйте автоматическую настройку системы
3. Документы будут обработаны и добавлены в базу данных с embeddings
